## 프로젝트 소개

한국섬유산업연합회에서 운영한 섬유·패션 산업 재직자 대상 교육 과정에서 활용된 파이썬 프로그래밍 및 웹 크롤링 실습 자료입니다.

본 프로젝트는 웹 데이터 수집 자동화의 기초 개념을 학습하고, 실제 웹사이트로부터 데이터를 수집·가공하는 과정을 실습하기 위해 제작되었습니다. 교육 과정에서는 Python 기반 데이터 처리, HTML 분석, 크롤링 자동화, 데이터 저장 및 활용 방법을 다루었습니다.

### 주요 내용

* Python 기초 문법
* 웹 크롤링 기초
* HTML 구조 분석
* BeautifulSoup 활용 데이터 수집
* Selenium 기반 동적 페이지 크롤링
* 데이터 저장 및 전처리
* 실습 예제 및 과제

### 교육 대상

* 섬유·패션 산업 재직자
* 데이터 수집 및 업무 자동화에 관심 있는 실무자
* Python 프로그래밍 입문자
